# Changelog

## v1.6.0 - 2026-04-11

### Added

- 普通文本消息支持一键复制，文本消息仍可继续加入收藏夹
- 单文件消息支持一键下载
- 多张图片可自动合并为一条图片组消息，支持展开 / 收起和整组下载
- 输入区新增剪贴板发送按钮，可直接发送剪贴板中的文本或图片
- 长文本消息支持自动折叠和手动展开
- 侧边栏补充了发送方式和消息操作说明

### Changed

- 后端消息模型新增 `message_type` 与图片组子项存储，历史记录、清空记录、物理清理逻辑已兼容图片组
- 图片组卡片重做为固定预览框，避免不同尺寸图片遮挡文字
- 触屏设备上的消息操作按钮改为常显，避免依赖 hover

### Notes

- 文本消息仍保持一发一条，本版本没有加入文本批量合并
- 图片组只支持纯图片批量发送，图片和其它文件仍需分开发送
